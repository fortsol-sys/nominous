# Project Spec

## Objective
A minimalist desktop application to show countdowns to important events. This will be one of the many micro-applications that I will be using to refine my productivity and workflows.

## Functional Requirements
- A timer to show countdowns (both graphical and numerical countdown options) to important events.
- To start with, a new event needs to be created by user and saved, which is immediately start a countdown in the UI.
- App should be able to track multiple events. Each saved event is added to a scrollable list of events on the left side of the UI.
- When any of the events is selected with a left mouse click, the right side of the UI should display time left (months/weeks/days/hours/minutes/seconds) in large font.
- Within pane, a button should be made available to log a quick comment. This comment should be added to the specific event's history.

## Technical Requirements
<!-- List technical constraints or stack choices -->
- The application should be cross platform and not browser based.
- The application needs to be as lightweight as possible.
- The data created by user should be saved in a folder as MD files.
- The UI displaying the countdown should be robust enough to switch between a simple countdown using a clock or display the history of the event as a time line with comments displayed as dots on the timeline.
- The UI should be extensible so that the data can be displayed in other formats/graphs/etc., in the future.
